## Methodology

1. **kf → HLS IDs:** Derived from SSRQ Fuseki RDF dump (`ssrq__fuseki_042810.ttl`) via `pers:refs [ pers:id "XXXXX" ; pers:type "HLS" ]`. 162 of 5,260 kf persons have HLS IDs; all confirmed as the same historical person (155 exact name match + 7 name variants confirmed by expert).

2. **HLS → EOS:** Each of the 162 kf persons with HLS IDs was searched in EOS HGB Basel's `spans` table (class='per', element='reference') using two strategies:
   - Exact text match against kf `main_name`
   - GLOB pattern match for forename + surname within the EOS span text

3. **Quality filtering:** Matches were filtered to:
   - **High (score=100):** Exact name match + EOS year falls within kf lifetime ±20 years (or kf lifetime unknown)
   - **Medium (score=50):** Name variant match (forename+surname present) + year overlap validated

4. **Known limitations:**
   - kf and EOS have narrow temporal overlap (kf: ~1305–1450; EOS: 1400–1700)
   - kf contains high-status Alemannic nobility largely absent from Basel's land-register corpus
   - Historical dates in kf are often approximate ("um", "ca.", "nach") — year tolerance is generous
   - 152 of 162 kf→HLS persons have no EOS match at all, which is expected given the above
